# Assignment 01 - Basic Data Exploration and Cleaning using Pandas

## Objective

The objective of this assignment was to learn the basics of data analysis using Pandas and perform data exploration and cleaning on a shopping dataset.

## Dataset

Shopping Dataset from Kaggle

## Tasks Performed

* Loaded the dataset into a Pandas DataFrame
* Explored the dataset using head(), tail(), shape, columns, and data types
* Identified and handled missing values
* Performed basic filtering and column selection
* Removed duplicate records
* Created a derived column (`total_amount`)
* Saved the cleaned dataset as a new CSV file

## Additional Analysis

Apart from the required tasks, I also:

* Cleaned price-related columns
* Calculated average ratings and discounts
* Analyzed product categories
* Created a popularity score using ratings and ratings count

## Files Included

* `Data_Exploration_and_Cleaning.ipynb` – Jupyter Notebook containing the complete workflow
* `Combined_dataset.csv` – Original dataset
* `cleaned_dataset.csv` – Cleaned dataset after preprocessing

## Tools Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

## Outcome

The dataset was successfully explored, cleaned, and transformed. The cleaned dataset was exported as a CSV file for further analysis.
